# 🤖 AI Chatbot with NLP

## 📝 Description
Built an AI chatbot using Natural Language 
Processing and Deep Learning with TensorFlow 
and NLTK in Python.

## 🧠 How it works
- User types a message
- NLTK tokenizes and stems the text
- Neural network predicts the intent
- Bot responds with appropriate answer

## 🤖 Models Used
- Deep Neural Network (TensorFlow)
- Natural Language Processing (NLTK)

## 📊 Intents Trained
| Intent | Example Input |
|---|---|
| Greeting | Hi, Hello, Hey |
| Goodbye | Bye, See you |
| Name | What is your name |
| Age | How old are you |
| Thanks | Thank you |
| About | What can you do |

## 🛠️ Tools Used
- Python
- TensorFlow
- NLTK
- NumPy
- Google Colab

## 🚀 How to Run
1. Open the .ipynb file in Google Colab
2. Run all cells in order
3. Type your message when prompted
4. Type quit to stop the chatbot
